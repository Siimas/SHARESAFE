# Share to a Group of Users

Sharing files could be done not only to all users but to a specific set of users, in wich we call "Groups". Every user can create a group and specify wich users will belong to that group.

## How to Create a Group

In order to create a group, the user must:

1. Go to groups page;
2. Click the "New Group" button;
3. Enter the group name;
4. Enter the user's email wished to be added to the group and click "Add Member" (do this for every user wished to be added);
5. Click "Create Group".

## How to Edit a Group

In edit mode, the user can leave a group or add members to the group.

### Add Member to Existing Group

In order to add member to existing group, the user must:

1. Click Edit on the desired group;
2. Type the desired user email wished to be added;
3. Click "Add Member".

### Leave Group

In order to leave a group, the user must click Edit on the desired group and then click the "Leave Group" button.

## How to Share a File with a Specifc Group

In order to share a file with a specific Group, the user must:

1. Go to groups page;
2. Click the desired group;
3. Click the "Send File" button;
4. Fill out the parameters;
5. Send file.
